CloudTune • проект в проде

Музыкальный сервис с локальным и облачным режимом

CloudTune - это рабочий fullstack: Flutter-клиент, backend на Go и PostgreSQL, плюс отдельный Telegram-бот для мониторинга. Приложение уже умеет локальную библиотеку, облачную синхронизацию, плейлисты, лайки и фоновое воспроизведение.

  • Backend: Go 1.24 + Gin

  • Monitoring: Python 3.12

Что внутри проекта

CloudTune сделан как единый стек: мобильный/desktop-клиент, API и инфраструктура мониторинга. Ниже ключевые функции, которые уже доступны в текущей версии.

JWT-аутентификация и персональная библиотека пользователя.
Дедупликация треков в облаке по content_hash.
Локальные и облачные плейлисты, включая системный Liked songs.
Фоновый плеер, queue, shuffle, repeat-one, управление через media session.
Windows desktop-shell с отдельной навигацией и правой player-панелью.
Telegram мониторинг: статус, хранилище, пользователи, авто-алерты.

Схема работы

Flutter App -> REST API
Go Backend -> PostgreSQL + uploads
Monitoring Bot -> /api/monitor/*
Оповещения и команды -> Telegram

Прод-окружение работает под Ubuntu 22.04: backend в Docker, Nginx как reverse proxy, SSL от Let's Encrypt.
